# Literary Lounge 📚

Literary Lounge is a modern and fully responsive digital library website built using HTML, CSS, and JavaScript. It allows users to explore books, search and filter collections, and enjoy a clean interactive shopping experience.

## Features ✨

* User Signup & Login system
* Dynamic book library using JavaScript
* Search, filter, and sort books
* Add books to favorites ❤️
* Dark & Light mode toggle 🌙☀️
* Responsive design for all devices
* Daily literary quotes using Fetch API

## JavaScript Concepts Used 💻

* DOM Manipulation
* Event Handling
* Arrays & Higher Order Functions
* Objects and JSON
* Local Storage
* Async/Await & Fetch API
* Promises
* Timers and Animations

## Tech Stack 🛠️

* HTML5
* CSS3
* JavaScript

## Project Goal 🎯

The main goal of this project is to create a visually appealing and interactive frontend website while demonstrating important JavaScript concepts learned in college.
